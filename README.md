# VidPrompt AI - Versi Terbaik

Video → Keyframes (pose + adegan change) → Prompt AI siap pakai (Kling, Runway, Luma, Veo)

**Fitur:**
- Deteksi pose change otomatis
- Deteksi adegan change
- Gallery keyframes interaktif
- Prompt generator profesional
- 100% lokal & gratis

Jalankan dengan: `streamlit run app.py`
